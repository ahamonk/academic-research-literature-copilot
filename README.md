# Academic Research Trend and Literature Review Copilot

An AI-powered web application that automatically tracks the latest research papers from arXiv, clusters them into emerging topics, and generates weekly literature reviews using a LangGraph multi-agent pipeline.

## Tech Stack

- Frontend: React + Tailwind CSS
- Backend: FastAPI
- Database: PostgreSQL
- Vector Database: ChromaDB
- Embeddings: Sentence Transformers
- Clustering: HDBSCAN
- Agent Framework: LangGraph
- Tool Layer: MCP Server
- Notifications: Slack API
